# EASYSTUDENTS

## Project Overview
EASYSTUDENTS is a project designed to facilitate student management and improve communication between students and educators. The application offers features such as student enrollment, course management, and performance tracking.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/A22SS/EASYSTUDENTS.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd EASYSTUDENTS
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the application:**
   ```bash
   npm start
   ```

Enjoy using EASYSTUDENTS!
